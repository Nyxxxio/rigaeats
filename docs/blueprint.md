# **App Name**: Singh's Spices

## Core Features:

- Reservation System: Allow users to book tables online, selecting date, time, and party size.
- Menu Display: Showcase the restaurant's menu with descriptions and prices.
- Image Gallery: Display an image gallery showing images of both the venue and plates.
- AI Recipe Suggestions: Suggest recipes that share key flavors and/or ingredients as dishes found on the menu, to inspire customers at home; this AI-powered tool generates recipe ideas with similar ingredients and/or flavors.
- Contact Information: Provide contact details, including phone number, email, and address with map integration.

## Style Guidelines:

- Primary color: Black (#000000) for a modern and elegant feel.
- Secondary color: White (#FFFFFF) to create a clean and simplistic design.
- Accent color: Gray (#808080) for subtle highlights and interactive elements.
- Body text: 'Roboto' (sans-serif), ensures readability with a modern touch.
- Headline text: 'Montserrat' (sans-serif), used to communicate an elegant, high-end feel with good contrast.
- Use minimalist line icons for a clean and modern visual appeal.
- A clean, single-page layout with clear sections for reservations, menu, and contact details, emphasizing whitespace.
- Subtle fade-in effects on menu items and interactive elements for a seamless user experience.
